# ATMStatusMonitoring
WPF приложение для оперирования состоянием банкоматов и передачи этой информации между структурными подразделениями банка.

В приложении используется `Entity Framework` и и база данных `ATMDatabase` для хранения информации о пользователях приложения, банкоматах и их состояниях.

Путь к базе данных указыватся в`app.config` как `ATMDb`
